Consejos de Ciberseguridad

Buenas Prácticas para Protegerse en Línea

1. Protege tus contraseñas: 2. Mantén tu software actualizado: 3. Ten cuidado con el phishing y el correo electrónico: 4. Utiliza un antivirus y un firewall: 5. Utiliza redes seguras: 6. Ten precaución con las descargas:
    Crea contraseñas robustas, difíciles de adivinar y diferentes para cada cuenta:
    Una contraseña robusta debe ser lo suficientemente compleja para evitar que alguien la adivine fácilmente o que pueda ser descifrada por programas automatizados (como ataques de fuerza bruta). Las características recomendadas incluyen:
    - Al menos 12 caracteres y combinar letras mayúsculas y minúsculas, números y símbolos.
    - Evitar palabras del diccionario o combinaciones comunes como “123456” o “password”.
    - Además, es importante que cada cuenta tenga una contraseña única.
    Usar la misma contraseña en múltiples sitios pone en riesgo todas tus cuentas si una de ellas se ve comprometida.
    Considera usar un administrador de contraseñas para almacenarlas de forma segura.
    Recordar muchas contraseñas complejas puede ser muy difícil. Para facilitar esto, puedes usar un administrador de contraseñas, que es una herramienta segura que guarda y organiza todas tus contraseñas en una bóveda cifrada. Estos programas pueden:
    - Generar contraseñas fuertes automáticamente.
    - Rellenar formularios de inicio de sesión por ti.
    - Alertarte si una de tus contraseñas aparece en una filtración de datos.
    Ejemplos populares: Bitwarden, 1Password, LastPass, KeePass, entre otros.
    No uses información personal fácilmente accesible como fechas de nacimiento o nombres de mascotas. Las contraseñas que contienen datos personales como tu nombre, fecha de nacimiento, nombre de tu mascota o lugar de residencia son muy débiles. Este tipo de información puede obtenerse fácilmente a través de redes sociales o búsquedas en línea. Un atacante que te conoce (o que busca información sobre ti) podría probar con estos datos primero.
    Evita también combinaciones como:
    - “Juan1985”
    - “Sofi123”
    - “Chile2024”
    Cambia tus contraseñas regularmente.
    Aunque ya uses contraseñas fuertes, cambiarlas periódicamente es una buena práctica para minimizar riesgos, especialmente si sospechas que una cuenta fue comprometida o si hubo una filtración de datos en algún servicio que usas.
    Consejos:
    - No uses versiones muy parecidas de tu contraseña anterior.
    - Considera activar la autenticación en dos pasos (2FA) para agregar una capa extra de seguridad.
    Asegúrate de que tu sistema operativo, navegadores y aplicaciones estén siempre actualizados con los últimos parches de seguridad.
    Los desarrolladores de sistemas operativos (como Windows, macOS, Linux), navegadores (como Chrome, Firefox, Edge) y aplicaciones (como Zoom, WhatsApp, Office, etc.) lanzan constantemente actualizaciones que corrigen vulnerabilidades de seguridad. Estas vulnerabilidades pueden ser aprovechadas por ciberatacantes para acceder a tus datos, instalar malware o controlar tu equipo sin que te des cuenta.

    Mantener tu software actualizado significa:
    - Protegerte contra nuevas amenazas.
    - Mejorar la estabilidad del sistema y el rendimiento.
    - Asegurarte de que tus dispositivos funcionen correctamente con los últimos estándares de seguridad.
    Habilita las actualizaciones automáticas cuando sea posible. Una de las mejores maneras de asegurarte de que todo tu software esté protegido es activar las actualizaciones automáticas. Esto te evita tener que preocuparte por revisar manualmente si hay nuevas versiones disponibles.

    Ventajas:
    - Recibes los parches de seguridad apenas se publican.
    - Reduces la probabilidad de que olvides actualizar una aplicación importante.
    - Te aseguras de que incluso programas que usas poco estén protegidos.

    Puedes habilitar esta opción en:
    - Sistema operativo: Windows Update, actualizaciones automáticas de macOS, etc.
    - Aplicaciones móviles: Desde los ajustes de Google Play o App Store.
    Navegadores y programas de escritorio: La mayoría tienen una opción para actualizarse automáticamente o al menos recordártelo.
    No hagas clic en enlaces sospechosos ni abras archivos adjuntos de remitentes desconocidos.
    Verifica la dirección del remitente y busca señales de phishing, como errores ortográficos o gramaticales.
    ¡¡¡No compartas tus datos personales!!!
    Instala un software antivirus y antimalware confiable y mantenlo actualizado.
    Configura un firewall para proteger tu red doméstica o empresarial.
    Evita las redes Wi-Fi públicas para transacciones financieras o acceso a información sensible.
    Si necesitas usar una red pública, considera usar una VPN para cifrar tus datos.
    Descarga software solo de fuentes confiables.
    Escanea los archivos descargados con un antivirus antes de abrirlos.
