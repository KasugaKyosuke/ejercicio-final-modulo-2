Ataques cibernéticos

¿Qué son?

Un ciber ataque o ataque cibernético, es un intento de obtener acceso, dañar o interrumpir sistemas informáticos, redes o dispositivos. Estos ataques pueden ser realizados por uno o varios individuos y con diversos fines, como lo son el robo de información, interrumpir servicios, robo de datos, fraude o espionaje y existen diversos tipos y técnicas con las que se llevan a cabo. A continuación, te contamos sobre ellas.

Malware

Software malicioso diseñado con el fin de infiltrarse en sistemas informáticos.

Virus

Programas maliciosos que se adhieren a archivos o programas legítimos y se propagan de un sistema a otro.

Gusanos

Programas maliciosos que se replican y propagan a través de redes sin necesidad de un archivo huésped.

Troyanos

Programas maliciosos que se disfrazan de software legítimo para engañar a los usuarios y obtener acceso a sus sistemas.

Spyware

Programas maliciosos que recopilan información sobre las actividades de los usuarios y la transmiten a terceros.

Ransomware

Tipo de malware que cifra los archivos de una víctima y exige un rescate para desbloquearlos.

Phishing

Ataques que utilizan correos electrónicos o mensajes fraudulentos para engañar a los usuarios y obtener información confidencial, como contraseñas, números de tarjetas de crédito, etc.

Spear Phishing

Ataques de phishing dirigidos a individuos específicos o grupos dentro de una organización.

Whaling

Ataques de phishing dirigidos a altos ejecutivos o personas con acceso a información valiosa.

Ataques de Denegación de Servicio (DoS/DDoS)

intento malicioso de interrumpir el funcionamiento normal de un sistema, servicio o red al inundarlo con tráfico ilegítimo, dejándolo inaccesible para sus usuarios legítimos

DoS

Ataques que sobrecargan un sistema o red con tráfico para que no pueda atender solicitudes legítimas.

DDoS

Ataques DoS que utilizan múltiples sistemas comprometidos (una botnet) para generar un volumen de tráfico aún mayor.

Otros tipos de ataques

Existen aún más tipos de ataques, algunos que buscan vulnerabilidades existentes en sitios web o aplicaciones y otros que no necesariamente están relacionados al uso de software, pero que aún así están enfocados en la búsqueda del acceso y robo de información.

Ataques Man-in-the-middle (MitM)

Ataques en los que un atacante se interpone entre dos partes que se comunican, interceptando y posiblemente modificando la comunicación.

Inyección SQL

Ataques que aprovechan vulnerabilidades en las aplicaciones web para insertar código SQL malicioso en una base de datos, lo que puede llevar a la manipulación o robo de datos.

Hacking de Contraseñas

Ataques que buscan obtener contraseñas mediante fuerza bruta, diccionario, ataques de ingeniería social o aprovechando vulnerabilidades en los sistemas.

Ataques de Fuerza Bruta

Ataques que intentan adivinar contraseñas o claves de acceso probando diferentes combinaciones hasta encontrar la correcta.

Ataques de Ingeniería Social

Ataques que manipulan a las personas para que realicen acciones o divulguen información confidencial, a menudo utilizando la psicología y la manipulación emocional.

Ataques de Intermediarios

Ataques que aprovechan la vulnerabilidad de las conexiones de red para interceptar o modificar la comunicación entre dos partes.

Ataques de CriptoJacking

Ataques que utilizan la potencia informática de un sistema comprometido para minar criptomonedas sin el consentimiento del usuario.

Ataques de Suplantación de Identidad (Spoofing)

Ataques que falsifican la identidad de una fuente legítima para engañar a los usuarios o sistemas.

Casos importantes y recientes de ataques cibernéticos

Caso Solarwinds (2020)

Un adversario del estado nación ruso obtuvo acceso a los sistemas de SolarWinds e implementó actualizaciones con troyanos para el software Orion.

Ataque a sony pictures (2014)

un ciberataque masivo atribuido a hackers norcoreanos, quienes filtraron datos confidenciales, incluyendo correos electrónicos, guiones, películas inéditas y detalles personales de empleados.

Caso Wannacry (2017)

Fue un ciberataque global de ransomware que afectó a miles de sistemas en más de 150 países, explotando una vulnerabilidad en Windows para cifrar datos y exigir rescates de bitcoin, causando perdidas millonarias y afectando hospitales, empresas y gobiernos.
